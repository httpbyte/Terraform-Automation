Within this repository I was able to utilize Terraform and AWS to automate infrastructure provisioning;

- Provisioned cloud infrastructure on AWS using Terraform with a modular Infrastructure as Code (IaC) architecture
- Configured AWS VPC with public subnets, internet gateway, and route tables using community Terraform modules
- Deployed and managed EC2 instances with automated bootstrapping via user_data shell scripting
- Implemented remote state management using AWS S3 backend for team collaboration and state locking
- Designed and enforced network security using AWS Security Groups with least-privilege ingress rules for SSH and HTTP traffic
- Containerized and deployed Nginx web server using Docker on cloud infrastructure
- Automated server configuration using Linux bash scripting and systemd service management
- Managed SSH key pairs and secure remote access to cloud instances
- Structured reusable Terraform modules for VPC networking and web server components, improving maintainability and scalability
- Utilized Git for version control with feature branching and remote repository management on GitHub
- Configured .gitignore to prevent sensitive files and large provider binaries from being committed to version control-
